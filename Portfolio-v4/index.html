    • That being said, some project I'm currently pretty proud of are:
      • Blueprint - Inspired by Bryan Johnson's Blueprint, this service uses LLMs to score university work. It retrieves new work from various sites, evaluates its time commitment, and schedules it into Google calendar. This removes the need for me to plan my day or decide what to do next. (Note: This program doesn't do the work for me, it simply estimates how long it would take me to do and then schedules it.) Find it here.
      • Formal-Deduction - A Python engine for parsing, and performing inference on propositional and first order statements. You can define stuff like \(((A \land \neg B) \lor \neg C) \rightarrow ((C \leftrightarrow B) \lor \neg A)\), and it can spit out a truth table, determine its semantic value, and let you perform syntactic transformations to it. You can also supply multiple such statements and have it determine if they are consistent, and if they are, give you a model that satisfies them. It's pretty cool. You can find it here.
      • ARM-LEG Simulator - Another Python project, this one simulates the execution of a subset of ARMv7 assembly. It can simulate the execution of a program along with the state of the registers and memory at each step. It can also simulate the execution of a program with a given set of inputs. You can find it on github.
